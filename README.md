# August & Oak

August & Oak is a modern and responsive e-commerce website built with HTML, CSS, and JavaScript. Designed with a clean, elegant interface, it delivers a seamless shopping experience across desktop, tablet, and mobile devices.

## Features

- Responsive and mobile-friendly design
- Modern and elegant user interface
- Featured collections and products
- Product search
- Category filtering
- Product sorting
- Shopping cart
- Wishlist
- Product quick view
- Quantity controls
- Local Storage support
- Newsletter subscription
- Customer testimonials
- FAQ section
- Contact section
- Smooth animations and transitions

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```
august-and-oak-ecommerce/
├── index.html

├── script.js
└── README.md
```

## Getting Started

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your preferred web browser.
